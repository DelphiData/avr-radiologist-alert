Setup checklist

  1. Repository: DelphiData/avr-radiologist-alert (created)
  2. Add repository secrets (Settings → Secrets and variables → Actions):
    • TWILIO_ACCOUNT_SID
    • TWILIO_AUTH_TOKEN
    • TWILIO_FROM_NUMBER (E.164, e.g., +16065551234)
    • AVR_USERNAME
    • AVR_PASSWORD
    • TIMEZONE (use America/New_York)
    Paste this snippet into your workflow env section:
    AVR_USERNAME: ${{ secrets.AVR_USERNAME }}
AVR_PASSWORD: ${{ secrets.AVR_PASSWORD }}
TWILIO_ACCOUNT_SID: ${{ secrets.TWILIO_ACCOUNT_SID }}
TWILIO_AUTH_TOKEN: ${{ secrets.TWILIO_AUTH_TOKEN }}
TWILIO_FROM_NUMBER: ${{ secrets.TWILIO_FROM_NUMBER }}
  3. Commit the monitoring files to the repo:
    • monitor.py
    • .github/workflows/monitor.yml
    • config.yml
    • contacts.yml
    • docs/index.html (this page)
  4. (Optional) Enable GitHub Pages to serve this status page from the docs folder.
